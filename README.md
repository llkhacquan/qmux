# qmux

Terminal-first development workflow tools for tmux.

## sidebar-go

Tmux sidebar that displays all Claude Code panes as a card dashboard with realtime status, git info, context %, and live tool intent. Single Go binary, daemon + thin-client architecture.

See [sidebar-go/README.md](sidebar-go/README.md) for setup, features, and usage.
